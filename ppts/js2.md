## 前言
----

* 网页的解剖。在我们深入了解JavaScript语言之前, 让我们解剖一下一个网页主要有哪些Web技术. {:&.build} 

* HTML 是内容:
         
    ```bash
             
         HTML是用来定义和描述的内容的标记语言。无论是博客文章，搜索引擎结果，
         或电子商务网站，一个网页的核心内容是用HTML编写的。
         HTML是用来描述内容的通用术语（标题，段落，图片等）  
                      
    ```

    ```bash
             
         CSS为HTML文档添加适当风格的附加语言。
    ```
